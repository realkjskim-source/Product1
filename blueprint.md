# Project Blueprint - Product1

## Overview
This is a modern web application built with vanilla HTML, CSS, and JavaScript, following the Firebase Studio AI Development Guidelines.

## Project Details
- **Architecture:** Framework-less, ES Modules.
- **Styling:** Modern CSS (Baseline) features with CSS Variables.
- **Entry Point:** `index.html`.

## Features
- **Lotto Number Generator:** Generates 6 random numbers (1-45).
- **Theme Switching:** Supports Dark Mode and Light Mode with persistent user preference.
- **Responsive Design:** Polished UI that works on various screen sizes.

## Implementation Details
- **Refactoring:** Internal styles and scripts moved to `style.css` and `main.js`.
- **Theme Implementation:** CSS variables and `data-theme` attribute for seamless switching.
- **Persistence:** `localStorage` integration for consistent experience across sessions.
- **Deployment:** Synchronized with GitHub repository.
