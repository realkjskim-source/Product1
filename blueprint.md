# Project Blueprint - Product1 (LottoPremium)

## Overview
LottoPremium은 고품질의 로또 번호 생성 및 정보 제공 서비스를 지향하는 웹 애플리케이션입니다. Google AdSense 승인 기준에 맞춰 설계되었으며, 풍부한 텍스트 콘텐츠와 명확한 사이트 구조를 갖추고 있습니다.

## Project Details
- **Architecture:** Framework-less (Vanilla JS), ES Modules.
- **Styling:** 시맨틱 HTML 및 CSS Variables를 이용한 프리미엄 반응형 디자인.
- **SEO:** 메타 태그 최적화 및 구조화된 콘텐츠 제공.

## 구현된 주요 기능
1. **로또 번호 생성기:** 무작위 추출 알고리즘과 애니메이션 효과.
2. **테마 전환:** 다크/라이트 모드 지원 및 설정 유지.
3. **콘텐츠 아티클:** 로또 관련 가이드, 통계, 건전한 복권 문화 등 텍스트 콘텐츠 제공 (AdSense 대응).
4. **필수 페이지:** 사이트 소개(About), 개인정보처리방침(Privacy Policy) 페이지 구축.
5. **외부 연동:** Formspree(문의), Disqus(댓글), Google AdSense(수익화).

## 업데이트 내역: AdSense 고품질 사이트 최적화
- **구조 개편:** Fixed Header, Footer, Main Section 등 시맨틱 태그 적용.
- **내비게이션:** 페이지 간 이동이 용이한 메뉴 시스템 구축.
- **콘텐츠 보강:** 3개 이상의 상세 안내 아티클 추가로 '콘텐츠 부족' 문제 해결.
- **법적 고지:** AdSense 필수 요건인 개인정보처리방침 명시.
