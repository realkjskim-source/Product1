# Project Blueprint - Product1

## Overview
프리미엄 로또 번호 생성기 웹 애플리케이션입니다. 바닐라 HTML, CSS, JavaScript를 사용하여 구축되었으며 현대적인 디자인과 다양한 사용자 기능을 포함하고 있습니다.

## Project Details
- **Architecture:** Framework-less (Vanilla JS), ES Modules.
- **Styling:** CSS Variables를 이용한 현대적이고 반응형인 디자인.
- **Persistence:** `localStorage`를 통한 사용자 설정 유지.

## 구현된 주요 기능
1. **로또 번호 생성기:** 1~45 사이의 중복 없는 6개 번호를 생성하고 정렬하여 애니메이션과 함께 표시합니다.
2. **테마 전환 (Dark/Light):** 다크 모드와 라이트 모드 전환 기능을 제공하며 사용자의 선택을 브라우저에 저장합니다.
3. **제휴 및 문의 폼:** Formspree API를 연동하여 사용자의 문의 사항을 이메일로 직접 받을 수 있습니다.
4. **댓글 시스템:** Disqus를 통합하여 사용자 간의 실시간 소통이 가능합니다.
5. **수익화:** Google AdSense 광고 스크립트를 통합하였습니다.

## 기술적 구현 사항
- **CSS Variables:** 테마별 색상 관리 및 부드러운 전환 효과.
- **Formspree:** 백엔드 코드 없이 폼 데이터 처리 및 이메일 전송.
- **Disqus:** 외부 댓글 서비스 연동.
- **Responsive Design:** 모든 기기에서 최적화된 레이아웃 제공.
